# ⚡ My Portfolio - Zuena

Website portofolio multi-halaman bertema **gamer futuristik** untuk tugas Pemrograman Web ITPLN.

### Halaman:
1. Home — Tentang diri dan skill (desain, editing, soft skills)
2. Portfolio — Galeri hasil desain dan foto editan
3. Contact — Form kerja sama proyek

### Fitur:
- Desain **responsive** (PC & HP)
- Efek **neon futuristik**
- Background berbeda tiap halaman
- Warna tegas (cyan, ungu, biru, magenta)
